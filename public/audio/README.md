# Ayush portfolio audio

## Primary score

The homepage uses Ludwig van Beethoven's **Piano Sonata No. 14 in C-sharp
minor, Op. 27 No. 2 — I. Adagio sostenuto**, performed by pianist **Paul
Pitman for Musopen**.

- `music/moonlight-adagio.ogg` is the locally hosted source recording.
- `music/moonlight-adagio.mp3` is a local 192 kbps compatibility transcode of
  the same recording.
- Duration: approximately 5 minutes 36 seconds.
- Recording status: **Public Domain (dedicated)**.

Licence and provenance were checked against the recording's displayed
metadata:

- Wikimedia Commons source and public-domain declaration:
  https://commons.wikimedia.org/wiki/File:Ludwig_van_Beethoven_-_sonata_no._14_in_c_sharp_minor_%27moonlight%27%2C_op._27_no._2_-_i._adagio_sostenuto.ogg
- IMSLP recording entry: Paul Pitman, Musopen, 2014, Public Domain (dedicated):
  https://imslp.org/wiki/Piano_Sonata_No.14,_Op.27_No.2_(Beethoven,_Ludwig_van)
- Musopen work page:
  https://musopen.org/music/2547-piano-sonata-no-14-in-c-sharp-minor-moonlight-sonata-op-27-no-2/

## Sound effects

No synthetic or oscillator-generated effects are used. Optional recorded Foley
files may be added later under `audio/sfx/` using the names documented in the
portfolio brief. Missing effects intentionally remain silent.
